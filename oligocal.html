Oligo Calculater

Oligo Calculater is a function written in python which calculat Tm for DNA Oligo, and generates reverse complementary sequence.

The input is the single DNA sequence not fasta format.

Currently, Oligo Calculater only can take one input sequece for calculation.

It will support multiple fasta input and fasta file soon.
